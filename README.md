# showtime

Supercharge your bash demo with this simple script.

### Installation

```bash
curl -O https://codeberg.org/inknos/showtime/raw/branch/main/showtime.sh
```

Then inside your demo

```bash
#!/bin/bash

source ./showtime.sh

p "Hello world!"
```
